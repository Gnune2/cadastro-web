# cadastro-web
cadastro web feito com js, html, css e uns frameworks
para rodar é necessário ter o node.js instalado
clonar o repositório
rodar no terminal bash (instalar as dependências):
npm install express  @prisma/client@6.19 bcrypt
npm install -D prisma@6.19 nodemon
iniciar o prisma
npx prisma init
configurar o arquivo .env com a string de conexão do banco de dados
npx prisma generate
